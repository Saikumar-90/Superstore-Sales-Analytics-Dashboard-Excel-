# Superstore-Sales-Analytics-Dashboard-2014-2017-

# Project Overview

Developed an interactive Sales Analytics Dashboard in Excel using the Superstore dataset (2014–2017). Performed data cleaning, feature engineering, and built Pivot Tables and charts to analyze sales, profit, discount impact, and trends. Implemented slicers for dynamic filtering and business insights.



## Key Features

* KPI cards for Total Sales, Total Profit, and Profit Margin
* Region-wise Sales & Profit analysis
* Category and Sub-category performance insights
* Discount impact analysis on profitability
* Time-based trend analysis (monthly & yearly)
* Interactive slicers (Region, Category, Segment)



## Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Feature Engineering
* Data Visualization
* Dashboard Design & Layouting



## Key Insights

* High discounts significantly reduce profit margins
* Technology category generates highest sales but also volatile profits
* Certain sub-categories consistently operate at a loss
* Seasonal trends observed with sales spikes in specific months
